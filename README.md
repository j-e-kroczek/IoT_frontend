## Node js
Project was created using Node js v20.11.0.

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```

### Linting & formatting your code.

```bash
    npm run lint
```
